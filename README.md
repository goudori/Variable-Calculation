## 🚩はじめに
### 文字列型と整数型の変数を使って、自分の名前と年齢をターミナルに表示してみました。

---



## 開発環境
- 使用OS:Windows11
- 使用言語:Java
- JDK: Windows版のOracle JDK 21.0.8 (LTS)　⚠️LTSサポート有効期間:2026年9月まで
- 使用IDE:IntelliJ IDEA Commnunity Edition(無料版)

---



## 利用方法
・Windowsの場合
「Shift + F10」のキーボードを同時に押して実行して下さい。



・Macの場合
「Control + R 」のキーボードを同時に押して実行して下さい。
